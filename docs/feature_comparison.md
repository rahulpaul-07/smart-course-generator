# Platform Feature Comparison

| Feature Category | AI-CourseGenerator (Competitor A) | Co-Teacher (Competitor B) | Text-to-Learn (Competitor C) | **Unified Course Platform** |
| :--- | :--- | :--- | :--- | :--- |
| **Course Generation** | Text only | Text + Links | Text only | **Text, YouTube, Flashcards, Quizzes** |
| **AI Speed** | ~45s (GPT-4) | ~30s (Claude 2) | ~10s (GPT-3.5) | **~3s (Groq Llama-3 70B)** |
| **Delivery Method** | Static Loading | Static Loading | Static Loading | **Real-time SSE Streaming** |
| **Agentic AI** | None | Basic Chatbot | None | **4-Agent Swarm (Review, Coach, Plan)** |
| **Adaptive Difficulty**| No | No | No | **Yes (Quiz-performance driven)** |
| **Collaboration** | No | Yes (Links) | No | **Yes (Profiles, Templates, Leaderboard)** |
| **Analytics** | None | Basic Completion | None | **Advanced (Streak, Heatmap, Weak Topics)** |
| **Architecture** | Monolith | Serverless | Monolith | **Micro-services ready, Dockerized** |

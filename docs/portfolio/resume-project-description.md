# Resume Project Description

**CourseAI Pro** | *Full-Stack LLM Learning Platform*
*Node.js, Express, React, MongoDB, Google Gemini, GitHub Actions*
- Architected and deployed a monolithic full-stack AI platform capable of generating interactive, heavily structured technical courses dynamically on-demand using Google's Gemini LLMs.
- Engineered a custom AI router with multi-provider fallbacks (Gemini, OpenRouter, Groq) and Server-Sent Events (SSE) streaming, decreasing initial Time-to-First-Byte latency by over 90% for end-users.
- Secured backend utilizing Zod validation schemas, Helmet headers, targeted rate limiters, and a server-side caching layer, achieving 95%+ repository quality scoring.
- Constructed a community marketplace with a gamified XP system, triggering automated public leaderboards via MongoDB aggregations.
